A lightbox that impresses

Your posters deserve a beautiful full-screen viewer. The built-in lightbox has everything you need:

  • Thumbnail navigation strip for quick browsing
  • Keyboard controls — arrow keys, Escape, Tab
  • Touch swipe gestures on mobile
  • Image preloading for instant transitions
  • Custom link button per poster (e.g. "Buy tickets")
  • Full accessibility: ARIA, focus trap, alt text

Powerful block editor integration

The Gutenberg block gives you full control without leaving the editor:

  • 20+ configurable settings in the sidebar panel
  • Add, edit, and delete posters inline
  • Drag & drop reorder in the editor preview
  • Set poster URL, orientation, alignment, and schedule
  • Customize "Show all" button appearance
  • Wide and full-width alignment support
